# Documentatie doorstroomopdracht

Deze map bevat deel 1 van de opdracht (analyse en ontwerp) voor het project **Gamebibliotheek**.

## Inhoud

- `analyse-ontwerp-gamebibliotheek.md`  
  Samenhangend analyse- en ontwerpdocument met:
  - analyse van eisen en wensen;
  - kwaliteitscriteria en validatie;
  - functioneel ontwerp;
  - interactieontwerp;
  - databaseontwerp (ERD, relationeel schema, DDL);
  - UML (usecase, usecasebeschrijving, klassendiagram, sequence, deployment);
  - advies.

- `sql/schema.sql`  
  SQL `CREATE TABLE` statements met foreign keys voor de eerste implementatie.

## Opmerking

Dit document is opgesteld zodat je het direct kunt gebruiken tijdens:

- inlevering van deel 1;
- voorbereiding op het assessment;
- start van deel 2 (realisatie).
